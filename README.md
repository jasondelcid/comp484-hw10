# comp484-hw10

## Description
This project is an updated version of the Giga Pet application created for COMP 484. Chrome DevTools features were implemented and demonstrated using console logging, browser debugging, error handling, breakpoints, watch expressions, and performance monitoring.



## Features
- Console logging examples
- Info, warning, and error messages
- Console tables and grouped logs
- Custom styled console messages
- 404 network error demonstration
- TypeError demonstration
- Violation/performance warning demonstration
- Message filtering in DevTools
- Breakpoints and debugging with the Sources panel
- Watch expressions and scope inspection
- Debugging and applying fixes to code



## Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
- Chrome DevTools
